# Task 4 - Modular Calculator (Java)

## Description
This project is a modular calculator application written in Java.
Each arithmetic operation is implemented as a separate method.

## Features
- Method overloading
- Modular design
- Exception handling for divide by zero
- Pass-by-value demonstration
- Reusable utility methods

## Technologies Used
- Java (JDK 17)
- Cursor / VS Code

## How to Run
1. Compile the program:
   javac ModularCalculator.java
2. Run the program:
   java ModularCalculator
